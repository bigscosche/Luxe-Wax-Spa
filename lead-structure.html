Supabase schema

Create this table in the Supabase SQL editor:

create table public.leads (
  id          uuid primary key default gen_random_uuid(),
  created_at  timestamptz not null default now(),
  name        text not null,
  phone       text not null,
  email       text not null,
  service     text not null,
  location    text not null,
  message     text,
  source      text,
  submitted   timestamptz
);

-- RLS: lock down everything. Netlify function uses service role key.
alter table public.leads enable row level security;

End-to-end flow

1. Visitor fills the form on /contact.html.
2. Submit handler POSTs JSON payload to /.netlify/functions/lead.
3. Netlify Function inserts a row into Supabase leads.
4. Same function POSTs a formatted message to a Slack Incoming Webhook.
5. Frontend shows the success message and resets the form.

Netlify Function (reference)

Save as /netlify/functions/lead.js. Set env vars in Netlify: SUPABASE_URL, SUPABASE_SERVICE_KEY, SLACK_WEBHOOK_URL.

export async function handler(event) {
  if (event.httpMethod !== 'POST') return { statusCode: 405 };

  const payload = JSON.parse(event.body || '{}');
  const required = ['name', 'phone', 'email', 'service', 'location'];
  for (const key of required) {
    if (!payload[key]) return { statusCode: 400, body: `missing ${key}` };
  }

  // 1. Insert into Supabase
  const dbRes = await fetch(`${process.env.SUPABASE_URL}/rest/v1/leads`, {
    method: 'POST',
    headers: {
      apikey: process.env.SUPABASE_SERVICE_KEY,
      Authorization: `Bearer ${process.env.SUPABASE_SERVICE_KEY}`,
      'Content-Type': 'application/json',
      Prefer: 'return=minimal',
    },
    body: JSON.stringify(payload),
  });
  if (!dbRes.ok) return { statusCode: 500, body: 'db_insert_failed' };

  // 2. Post to Slack
  await fetch(process.env.SLACK_WEBHOOK_URL, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      text: `*New lead — ${payload.service}* in ${payload.location}\n` +
            `${payload.name} · ${payload.phone} · ${payload.email}\n` +
            (payload.message ? `> ${payload.message}\n` : '') +
            `source: ${payload.source}`,
    }),
  });

  return { statusCode: 200, body: JSON.stringify({ ok: true }) };
}

Environment setup checklist

  1. Create a free Supabase project. Run the SQL above in the SQL editor.
  2. Copy the project URL and the service key (not the anon key — inserts happen server-side).
  3. Create a Slack Incoming Webhook for the target channel (#leads).
  4. In Netlify → Site settings → Environment variables, add SUPABASE_URL, SUPABASE_SERVICE_KEY, SLACK_WEBHOOK_URL.
  5. Deploy. Submit a test lead. Verify one row in Supabase and one message in Slack.
